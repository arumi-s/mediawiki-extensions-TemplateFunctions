# TemplateFunctions v1.1.0

This MediaWiki extension provides parser functiions that can be used in templates.

## Parser Functiions

- `{{#param:}}`
- `{{#getparam:}}`
- `{{#rawparam:}}`
- `{{#exeparam:}}`
- `{{#link:}}`
- `{{#addlink:}}`
- `{{#inoutro:}}`
- `{{#htmlencode:}}`
- `{{#htmldecode:}}`
- `{{#urlencodequery:}}`
- `{{#convertspec:}}`
